## 四、推荐问题设计原则

1. **覆盖高频需求**：每个角色的4个问题都是该角色最常咨询的内容
2. **引导性强**：问题表述清晰，能引导用户快速找到所需信息
3. **资料匹配**：每个问题都能在现有资料中找到准确答案
4. **简洁明了**：问题简短，易于理解和选择

## 五、推荐问题展示方式

在系统中，推荐问题以编号列表形式展示，用户输入编号即可快速提问：

```
💡 推荐问题（点击数字提问）：
  1. 报到流程是怎样的？需要带什么材料？
  2. 学费什么时候交？怎么交？
  3. 宿舍怎么分配？什么时候能入住？
  4. 校园卡在哪里办？有什么功能？
  0. 返回上级菜单
```
