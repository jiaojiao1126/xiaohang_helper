# 应急防骗指南

## 一、心理危机干预

### 1.1 识别心理危机信号
- 言语信号：说"活着没意思"、"不想活了"、"想死"等
- 行为信号：突然把重要物品送人、与人告别、情绪异常低落
- 情绪信号：持续抑郁、焦虑、绝望、易怒

### 1.2 求助方式
- **心理援助热线**：12320-5（全国心理援助热线）
- **学校心理咨询中心**：0371-61912356（学生活动中心3楼）
- **辅导员**：第一时间联系辅导员
- **医院急诊**：如有自伤行为，立即拨打120或前往校医院

### 1.3 求助原则
- 立即行动，不要犹豫
- 认真对待每一个求助信号
- 保护求助者隐私

## 二、防骗指南

### 2.1 常见诈骗类型

#### （1）冒充老师/辅导员诈骗
- 诈骗手法：冒充辅导员或老师，以"交学费"、"资料费"、"培训费"为由要求转账
- 防范要点：
  - 核实身份：通过官方渠道联系辅导员确认
  - 不轻易转账：任何要求转账的都要先核实

#### （2）冒充公检法诈骗
- 诈骗手法：冒充警察、法院工作人员，以"涉嫌洗钱"、"账户异常"为由要求转账到"安全账户"
- 防范要点：
  - 公检法机关不会电话办案
  - 不存在"安全账户"
  - 不透露银行卡信息和密码

#### （3）兼职刷单诈骗
- 诈骗手法：以"刷单返利"、"点赞赚钱"为由，先小额返利后大额骗取
- 防范要点：
  - 刷单是违法行为
  - 任何要求垫资的兼职都是诈骗

#### （4）网络贷款诈骗
- 诈骗手法：以"低利率"、"秒批"为由，要求先交"手续费"、"保证金"
- 防范要点：
  - 通过正规渠道贷款
  - 不预付任何费用

#### （5）中奖/退款诈骗
- 诈骗手法：以"中奖"、"商品退款"、"快递丢失"为由，要求提供银行卡信息或转账
- 防范要点：
  - 不贪小便宜
  - 官方渠道核实

### 2.2 防骗"六不"原则
1. 不轻信：不轻易相信陌生电话、短信、网络信息
2. 不透露：不透露个人信息、银行卡信息、验证码
3. 不转账：不轻易转账，尤其是陌生人要求的转账
4. 不扫码：不随意扫描不明二维码
5. 不链接：不点击不明链接
6. 不贪利：不贪小便宜，天上不会掉馅饼

### 2.3 被骗后处理
1. 立即停止转账
2. 保留证据（聊天记录、转账凭证）
3. 拨打110报警
4. 联系银行冻结账户
5. 向学校保卫处报告

## 三、校园安全

### 3.1 消防安全
- 禁止在宿舍使用大功率电器（电磁炉、电热水壶等）
- 禁止私拉乱接电线
- 熟悉消防通道和灭火器位置
- 发现火情立即拨打119和0371-61911199

### 3.2 财产安全
- 贵重物品妥善保管
- 离开宿舍锁好门窗
- 校园卡丢失立即挂失
- 不在宿舍存放大量现金

### 3.3 人身安全
- 夜间出行结伴而行
- 不走偏僻小路
- 遇到可疑人员及时报警
- 遵守交通规则
